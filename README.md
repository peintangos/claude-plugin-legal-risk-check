# Legal Risk Check - Claude Code Plugin

プロダクトの法的リスクを予備調査し、構造化されたレポートを自動生成する Claude Code プラグイン。

## 概要

ソフトウェアプロダクトのリリース前に確認すべき法的リスクを、7カテゴリ・5段階リスクレベルで自動スクリーニングします。日本法に特化しており、法令条文・判例・費用目安を含む実務的なレポートを生成します。

## 調査カテゴリ

| #   | カテゴリ                   | 主な調査内容                               |
| --- | -------------------------- | ------------------------------------------ |
| 1   | 知的財産権                 | 商標リスク、AI生成コンテンツの著作権帰属   |
| 2   | OSSライセンス              | 推移的依存を含む全件調査、コピーレフト検出 |
| 3   | プライバシー・個人情報保護 | 外部送信規律、Cookie、GDPR                 |
| 4   | 利用規約・API ToS          | サードパーティAPI/サービスのToS確認        |
| 5   | 特商法・電子商取引         | 特定商取引法、景品表示法                   |
| 6   | コンテンツ規制             | 著作権（複製権・翻案権）、AI利用表示       |
| 7   | 競争法                     | ステマ規制、景表法                         |

## リスクレベル

| レベル   | 意味                                 |
| -------- | ------------------------------------ |
| CRITICAL | 明確な法令違反・重大なライセンス違反 |
| HIGH     | 訴訟・行政処分のリスクあり           |
| MEDIUM   | グレーゾーン・条件付きリスク         |
| LOW      | ベストプラクティスからの逸脱         |
| INFO     | 問題なし / 参考情報                  |

## 特徴

- **コード実態の優先**: ユーザー回答とコードの実態が矛盾する場合、コードの実態に基づいてリスク評価
- **推移的依存の全件調査**: `license-checker` 等で数百件の間接依存パッケージも調査
- **部分リクエスト対応**: 「OSSライセンスだけチェック」等の特定カテゴリへの集中調査が可能
- **ディープリサーチ資料参照**: 3つのAIディープリサーチによる法務ガイドを参照し、法令条文・判例・費用目安をレポートに反映
- **構造化されたレポート**: サマリーテーブル + カテゴリ別詳細 + 優先度付きアクション一覧

## インストール

```bash
/plugin marketplace add peintangos/claude-plugin-legal-risk-check
/plugin install legal-risk-check@matsuojumpei-plugins
```

## 使い方

```
/legal-risk-check
```

または自然言語で:

- 「このプロジェクトの法的リスクを調査して」
- 「OSSライセンスに問題がないかチェックして」
- 「著作権的に問題ないか確認したい」

## 出力

`LEGAL-RISK-REPORT.md` がプロジェクトルートに生成されます。

## 免責事項

このプラグインはAIによる予備的なスクリーニングツールであり、法的助言ではありません。検出されたリスクの最終判断は、必ず弁護士等の法律専門家にご相談ください。
